# [/] BUFF_VRadio

> ## What is VRadio?
>> VRadio is a simple discord radio bot to search and play music in voice channel

> ## How to use the bot ?
>> - /arise : Join the voice channel
>> - /search [song_name] : Search a song
>> - /play [song_name] : Play the requested song or add the requested song to the song queue
>> - /stop : Stop the song
>> - /pause : Pause the song
>> - /skip : Skip the current song
>> - /release : Stop the song and leave the voice channel
>> - /autoplay : Toggle the autoplay
